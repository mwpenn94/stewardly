**Chosen: "Editorial Command Center"** — A dark, information-dense editorial aesthetic inspired by Bloomberg Terminal, The Information, and contemporary AI research reports (Anthropic, Inflection).

- Design Movement: Neo-editorial / data-journalism. Serif display headlines paired with monospaced numerics, grounded in a dark navy canvas.
- Core Principles: (1) Information density without clutter, (2) editorial typographic hierarchy, (3) amber accent as the single "voice" color, (4) real motion only where it aids comprehension.
- Color Philosophy: Deep navy canvas (#0B1F44), carbon panels (#0F172A/#111827), luminous off-white text, a single amber accent (#E07B00) for metrics-that-matter and an electric cyan (#38BDF8) for agent-related data trails. Emotional intent: authoritative, calm, premium.
- Layout Paradigm: Two-column asymmetric grid. Left sticky rail carries navigation and context; right canvas holds the hero, metric strips, charts, architecture, and the live agent demo. Numbers float in oversized display type, never boxed.
- Signature Elements: (1) thin amber hairlines as section dividers, (2) a recurring hexagon "agent node" motif that echoes the hero, (3) animated percentage counters.
- Interaction Philosophy: Hovers reveal data footnotes; the agent demo plays steps in real time; nothing jitters unnecessarily.
- Animation: Count-up on metrics when entering viewport; staggered fade-in for cards; the agent-demo "thinks, acts, reflects" with per-step reveal timed at 600–900ms.
- Typography System: **Fraunces** (display serif) for headlines, **Inter** for body, **JetBrains Mono** for numerics and the live agent log.
